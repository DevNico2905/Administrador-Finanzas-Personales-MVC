# 💰 Gestor de Gastos Personales (Expense Tracker) - .NET MVC

Este es un proyecto de portafolio basado en una aplicación de seguimiento de gastos. Permite gestionar categorías, registrar transacciones y visualizar estadísticas financieras mediante un panel moderno.

## 🚀 Características
- **CRUD Completo:** Gestión de Categorías y Transacciones.
- **Dashboard Visual:** Gráficos estadísticos utilizando **Syncfusion**.
- **Arquitectura:** Modelo-Vista-Controlador (MVC).
- **Entorno Multiplataforma:** Configurado específicamente para funcionar en **macOS**.

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** C# / .NET 
- **Framework:** ASP.NET Core MVC
- **ORM:** Entity Framework Core
- **Base de Datos:** SQL Server corriendo en **Docker** (Azure SQL Edge para Mac)
- **UI:** Syncfusion ASP.NET Core Controls
- **Editor:** Visual Studio Code
